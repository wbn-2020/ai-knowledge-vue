# KnowFlow AI Frontend

KnowFlow AI 个人知识库与智能文档问答平台前端工程。

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Element Plus
- Pinia
- Vue Router
- SCSS

## Pages

- 前台：登录注册、工作台、知识库管理、文档管理、智能问答、问答历史、语义搜索、用户设置
- 后台：数据看板、用户管理、知识库管理、文档管理、任务管理、AI 配置、Prompt 模板、日志审计、系统参数

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
